# yunyingbotv2

Web3 项目运营分析平台第一阶段骨架。

当前重点：

- 分析层 MVP
- AI 主收集 + 主分析
- 单因子复核与重算
- Prompt 与配置驱动

## 目录

- `apps/`：应用入口
- `packages/`：领域与应用模块
- `configs/`：因子、维度、工作流、评分等配置
- `prompts/`：Prompt 模板
- `docs/`：文档沉淀
- `scripts/`：辅助脚本
- `data/`：样例和 fixtures
- `infra/`：基础设施预留
